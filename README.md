# TinyLearn - Learning Management System

A modern, full-stack Learning Management System built with Laravel and React, featuring real-time notifications, role-based access control, and comprehensive course management.

## 🎯 Features

### Core Features
- **User Authentication** - Secure login/registration with Laravel Sanctum
- **Role-Based Access Control** - Admin, Teacher, and Student roles
- **Course Management** - Create, edit, and manage courses
- **Assignment System** - Create assignments with file submissions
